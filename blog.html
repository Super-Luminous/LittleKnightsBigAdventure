2026-07-16: Progression Update

A new update is live! A lot of new features have been added, and it is starting to feel like a proper game. I have also spent some time refactoring the code behind the scenes in order to fix some 'spaghetti code' issues, which will make it easier to add features and fix bugs in the future.

New Additions

  • Added a pause menu.
  • Next level will load once you finish the current level.
  • Added lives counter, will game over if you lose all your lives.
  • Added some basic animations.

Bug Fixes

  • Updated the checkpoint system and fixed some respawn bugs
  • Adjusted level generation code to fix some generation bugs

Work In Progess

  • Sounds!
  • Autum themed level.

This is the first update! I have made the game into a playable state as a prototype.Right now the focus is getting the gameplay down, feeling good, and free of bugs.

work in progess at the moment:
